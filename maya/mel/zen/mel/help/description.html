*ZenTools is no longer tested with versions of Maya preceeding Maya 2008.

Read ZenTools Documentation

  • Maya 8.5 or later is required to use ZenTools.
  • Bonus tools for Maya is required to use ZenTools. Bonus Tools can be found at http://area.autodesk.com/index.php/misc/bonus_tools/.

Modeling

  • Distribute edge loops along dynamically created curves from a vertex selection.
  • Distribute vertices over dynamically created surfaces lofted from edge selections.
  • Create history-based nurbs surfaces from any rectangular patch of polygon faces.
  • Create history-based nurbs curves from contiguous polygon edges.

Selection

  • Shortest edge path selection from any number of selected vertices.
  • Flood Select polygon vertices or faces.
  • Transfer Component Selection from one object to another.

UV Layout

  • Create nurbs-style parameterized uv patches on any rectangular patch of polygon faces.
  • Distribute uvs along curves created dynamically between uv selections.
  • Mirror UVs.

Hair

  • Create curve spans or Shave hairs from any rectangular patch of polygon faces.

Character Setup

  • Mirror Smooth Skin Influence Weights which have non-joint influence objects.
  • Easily Confine Skin Weights to specific influences.
